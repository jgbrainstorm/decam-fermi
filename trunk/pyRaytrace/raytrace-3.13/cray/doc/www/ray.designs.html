The data directory contains a large number of optical designs.

Here are a few of the more interesting ones:

  • 3.5m.dat - A simple description of the 3.5m telescope.
  • fred.bk7lf5 - A focal reducer design for use with the drift scan camera on the 3.5 m telescope.
  • SDSS Imager The current 2.5m design for imaging
  • SDSS Spectro The current 2.5m design for spectroscopic mode
  • An achromatic lens
  • TMAA63 design for SNAP
  • Simple Cassegrain design

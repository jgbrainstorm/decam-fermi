Definitions of all parameters

Note that every parmeter has a corresponding adjust flag.  Not all are
useful.

Focal plane mapping parameters
Only flags 1-99 are of use.

fplane	Param	Usage
Index	Index

0	Not used

1	1-99	Field Y center for filter I (mm)
1	101-199	Field Y width for filter I (mm)

2	1-99	Field X center for filter I (mm)
2	101-199	Field X width for filter I (mm)

3	1-99	Scale factor for filter I
3	101-199	Design wavelength for filter I

4	1-99	Distortion correction for filter I
4	101-199	0 = use normal mapping focal plane -> sky
			1 = use "drift scan" mapping

5	1-99	Rotation angle for filter I
5	101-199	Weight

6	1-99	Paraxial focal length (derived)
6	100-199	Paraxial exit pupil z position (derived)

optic
index

0	1-99	In principal, same meaning as for optical surface parameters.
		In practice, only 5 (Z position) is used to store the
		"point at infinity" z distance.
0	100-199	Initial refraction index for filter I

For all surface, parameter indices 200-399 are used for
parameter increments for least squares.  Only 200-299 are
of use.

For surface 0 only, flags 0-99 are used to signal that the least squares
program should use this color in the least squares fit.

Unnumbered structure members:

idint		Integer part of surface id (default is surface index)
idfract		Fraction part of surface id

Unnumbered derived parameters
q1		For strut stop, gives quadrant of p1-p2 line segment
q2		Ditto for p2-p3
q3		Ditto for p3-p4
q4		Ditto for p4-p1

For all surfaces, permissible flags are 1-199.
